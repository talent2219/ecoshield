=======================================
  ECOSHIELD - SETUP INSTRUCTIONS
=======================================

STEP 1 — Install backend packages
----------------------------------
Open terminal, go into the backend folder:
  cd backend
  npm install
  cd ..

STEP 2 — Install frontend packages
------------------------------------
  cd frontend
  npm install
  cd ..

STEP 3 — Run the app (need 2 terminals)
----------------------------------------
Terminal 1 (Backend):
  cd backend
  node server.js
  --> Should say: EcoShield running on port 5000

Terminal 2 (Frontend):
  cd frontend
  npm start
  --> Opens browser at http://localhost:3000
